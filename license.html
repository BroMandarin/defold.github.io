License inclusion

The Defold License specifies that you must include the license itself with all copies of Defold and in any derived work created using Defold. It is up to you to decide how you wish to distribute the license. Here are some examples of how this can be done:

  • Show the license at the end of a credits screen if your application has one
  • Show the license from a dedicated license screen or popup in your application
  • Print the license to an output log when your application starts
  • Put the license in a text file and include it with the distribution of your application
  • Put the license in a printed manual included with your application
