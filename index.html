No setup

Defold comes fully features out of the box. There's nothing to set up or configure, not even when bundling. Simply download the editor and get creative!

  • Visual Editor
  • Code Editor
  • Lua Debugger
  • Scene Editor
  • Particle Editor
  • Tilemap Editor

Truly cross-platform

Using just one code base, and without the need for external tools, Defold lets you publish your game to a multitude of platforms, including:

  • HTML5
  • Android
  • iOS
  • Steam
  • FB Instant Games
  • Windows
  • macOS
  • Linux

Easy Device Testing with Hot Reload

Work on your computer, test on your device. All changes you make can be reloaded and instantly previewed on your device. No need to rebuild and reinstall.
